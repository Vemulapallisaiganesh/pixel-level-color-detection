🎨 About

Pixel is an advanced image recognition and scene understanding platform that combines pixel-level color detection with deep learning-based object segmentation using YOLOv8.

⚡ Core Technology

  • YOLOv8 Segmentation: State-of-the-art real-time segmentation
  • OpenCV: High-performance image processing
  • Ultralytics: Modern deep learning framework

🔧 Key Features

  • Pixel-level color detection
  • Real-time object segmentation
  • Adjustable confidence thresholds
  • Batch image processing
  • Multi-format export

Capabilities

Object Detection

Detect and segment objects in images with high accuracy using YOLOv8.

Color Mapping

Apply vibrant colors to detected regions with automatic color selection.

Batch Processing

Process multiple images efficiently with customizable parameters.

Custom Training

Train models on custom datasets including COCO 2017 for specialized tasks.

Real-time Processing

Fast inference with adjustable confidence thresholds for different use cases.

Export Support

Save results in JPG, PNG, BMP, WebP and other formats.

Architecture Diagram (Folder Process)

uploads/ and input/

  • Image files are received
  • Upload API stores source images
  • Input files prepared for processing
→

app.py (API Layer)

  • /api/upload handles upload
  • /api/process runs inference
  • /output and /api/download serve results
→

main.py (Core Processing)

  • YOLOv8 segmentation inference
  • Mask extraction and contour operations
  • Pixel-level color filling
→

utils/color_utils.py

  • Color palette logic
  • Class-to-color mapping
  • Consistent color assignment

yolov8m-seg.pt (Model)

  • Pretrained segmentation weights
  • Object and mask prediction
  • Confidence-based filtering

templates/ + gui.py

  • home.html and processor.html UI
  • Flask web dashboard on localhost
  • Tkinter GUI alternative in gui.py

output/ (Final Result)

  • Processed result images
  • Preview image for dashboard
  • Download-ready output files
